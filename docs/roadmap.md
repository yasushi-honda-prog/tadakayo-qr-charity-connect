# 段階的開発ロードマップ

## 第1段階: 導入準備（〜1ヶ月）

- [ ] つながる募金への申し込み
  - 規約確認、申請フォーム入力、団体ロゴ提出
  - 審査期間: 約1ヶ月
- [ ] 既存HPへの寄付導線設計
  - 寄付案内ページの構成検討
  - 「PayPayで寄付する」ボタンの配置検討

## 第2段階: 導入・公開

- [ ] 既存HPに寄付ページ追加
  - 団体紹介、寄付の使い道を記載
  - つながる募金へのリンク設置
- [ ] GA4設定
  - 流入元パラメータ（?source=xxx）の計測設定
- [ ] QRコード作成
  - チラシ用、名刺用、イベント用など種類別に作成
- [ ] テスト寄付の実施
  - 実際のフローを確認

## 第3段階: 運用開始

- [ ] チラシ・名刺へのQRコード掲載
- [ ] イベントでの告知開始
- [ ] 寄付者への領収書発行フロー確立
  - つながる募金管理画面から寄付者情報取得
  - 領収書テンプレート作成

## 第4段階: 改善・拡張（寄付規模拡大時）

- [ ] 流入元別の寄付状況分析
- [ ] 効果的なQRコード配置の検討
- [ ] コングラントへの移行検討（年間寄付50万円超の場合）
  - 領収書自動発行
  - 寄付者CRM機能

## 継続的改善

- [ ] 寄付者へのお礼・報告の仕組み構築
- [ ] 寄付キャンペーンの企画・実施
- [ ] 認定NPO取得後の寄付金控除案内整備
