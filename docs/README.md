# タダカヨ QRチャリティ・コネクト ドキュメント

GitHub: `tadakayo-qr-charity-connect`
GCP: `tadakayo-qr-connect`

本ディレクトリは、AI駆動開発で継続的に設計を最適化しながら進めるための設計・運用ドキュメント集です。PayPay for Developers / 楽天ペイのAPI連携を前提とした設計を記載します。

## 目次

- 詳細技術スタック: `docs/tech-stack.md`
- API仕様（内部）: `docs/api-spec.md`
- データモデル: `docs/data-model.md`
- 環境変数・シークレット一覧: `docs/env-secrets.md`
- 監視・アラート: `docs/monitoring.md`
- テスト計画: `docs/testing.md`
- ADR一覧: `docs/adr/`
- 開発ロードマップ: `docs/roadmap.md`
- 運用上の注意点: `docs/operations.md`
- AI駆動開発ガイド: `docs/ai-development.md`

## 使い方
- 重要な設計変更はADRに記録し、`docs/adr/`へ追加します。
- 仕様・運用・セキュリティの変更があれば該当ドキュメントを更新します。
- ロードマップの進捗は四半期またはリリース単位で更新します。
