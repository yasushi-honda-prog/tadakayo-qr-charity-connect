# 決済プロバイダ申請ガイド

PayPay for Developers / 楽天ペイ（オンライン決済）への加盟店申請に必要な情報をまとめる。

## 申請の前提

タダカヨの「支援」は**対価性のある取引（売上）**として申請する。
- 研修の提供
- グッズの提供
- 活動支援（対価性あり）

## PayPay for Developers

### 必要情報（法人）

法人の場合、**書類アップロードは基本的に不要**。

| 項目 | 内容 | 準備状況 |
|------|------|----------|
| 法人名 | NPO法人タダカヨ | [ ] |
| 代表者氏名 | | [ ] |
| 連絡先メールアドレス | | [ ] |
| 連絡先電話番号 | | [ ] |
| 振込先銀行口座 | | [ ] |
| 事業内容 | 研修・グッズ提供等 | [ ] |

### 申請フロー

1. [PayPay for Developers](https://paypay.ne.jp/store-online/) で仮登録
2. 本登録案内メールを受信（`@paypay-corp.co.jp` を受信許可）
3. 審査情報を入力
4. 審査完了後、専用IDが発行される

### 注意事項

- 特定継続的役務（美容医療、エステ等）は追加書類が必要
- NPO法人としての事業説明を明確に

## 楽天ペイ（オンライン決済）

### 必要書類（法人）

| 書類 | 要件 | 準備状況 |
|------|------|----------|
| 履歴事項全部証明書（登記簿謄本） | 発行日から3か月以内 | [ ] |
| 法人番号 | [国税庁法人番号公表サイト](https://www.houjin-bangou.nta.go.jp/)で確認 | [ ] |
| 振込先口座情報 | 法人名義と同一 | [ ] |
| 代表者確認書類 | 登記簿と代表者が異なる場合のみ | [ ] |
| ホームページURL | タダカヨ公式サイト | [ ] |

### 申請フロー

1. [楽天ペイ（オンライン決済）](https://checkout.rakuten.co.jp/biz/) で申し込み
2. 必要書類をアップロード（鮮明な画像）
3. 審査（Visa/Mastercard: 3〜4営業日、その他: 約2週間）
4. 審査完了後、利用開始

### 注意事項

- 楽天銀行を振込口座に設定すると本人確認書類が不要
- 書類が不鮮明だと再提出になる

### 事業説明

申請時に記載する事業内容の例:

```
NPO法人タダカヨは、介護・福祉分野のICT活用を支援する団体です。
本決済システムでは以下のサービス・商品を提供します：
- 介護ICT研修の受講料
- オリジナルグッズの販売
- 活動支援金（研修資料の提供等の対価）
```

### Webhook受信要件

決済プロバイダからのWebhook受信に必要な情報:

| 項目 | 値 | 備考 |
|------|-----|------|
| Webhook URL | `https://xxx.run.app/webhook/paypay` | Cloud Run構築後に確定 |
| 固定IP | Cloud NAT経由で確保予定 | IP制限が必要な場合 |
